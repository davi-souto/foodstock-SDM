Organização XP
Main Description

Em todos, e principalmente nos projetos menores, as pessoas envolvidas em um projeto de software farão parte de uma ou mais entidades (empresas, grupos funcionais, governos, comunidades, etc.). As Práticas XP focam nos papéis diretamente envolvidos na identificação do que o software tem que fazer (Equipe Cliente XP) e no desenvolvimento desse software (Equipe Desenvolvedora XP). Para suportar estas equipes, Toda a Equipe inclui um terceiro grupo importante, a Organização XP.

Os papéis específicos que a XP identifica como partes da Organização XP são o Rastreador XP e o Treinador XP. A Organização XP também inclui todas as pessoas que compõem a infraestrutura que permite que o projeto exista. Isto inclui gestão, contabilidade, suporte, instalações, etc. Devido a esses papéis serem de suporte, a XP tenta minimizar a dependência da Equipe Cliente XP e da Equipe Desenvolvedor XP da Organização XP. Na XP, a meta é que toda a equipe seja auto-gestora e, em grande parte, autossuficiente. XP não fornece diretrizes práticas específicas para esses outros papéis, mas sugere que esses papéis também orientem as suas práticas com os Valores XP de comunicação, simplicidade, feedback e coragem.
