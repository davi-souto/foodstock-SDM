Main Description

A Programação Extrema baseia-se principalmente nos valores de comunicação, simplicidade, feedback e coragem. Estas não são somente palavras-do-momento: elas permeiam o comportamento dos profissionais XP e as escolha das práticas XP.

Mais do que qualquer outra coisa, XP baseia-se em pessoas, pessoas que ficam juntas e trabalham em conjunto para construir software. As práticas XP existem para permitir que as pessoas trabalhem em conjunto; as práticas tentam nunca interferir no empreendimento humano de construção de software que atenda às necessidades de negócio. Ou seja, as práticas XP, embora bastante disciplinadas, existem para permitir interações entre indivíduos, nunca para substituir ou interferir essas interações.
